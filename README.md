#   Banking Application Project

이번 프로젝트는 Oracle DB를 활용해, 고객,계좌,개별 이체 건에 대한 CRUD서비스와  register(회원가입),login(로그인),makeAcocunt(계좌개설),transaction(계좌이체) 서비스를 포함하였습니다.
